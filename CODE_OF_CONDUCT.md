# Code of Conduct

This project adopts the **Contributor Covenant**, version 2.1, as its code of conduct.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: be respectful and constructive, assume good faith, and help keep this a welcoming space
for everyone. Unacceptable behavior can be reported confidentially to the maintainer at
**groundhalt@gmail.com**. Maintainers may remove, edit, or reject contributions and interactions
that do not align with this code of conduct.
